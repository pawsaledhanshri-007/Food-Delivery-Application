# Food-Delivery-Application

This application connects restaurants, customers, and delivery partners on a single platform. Restaurants can register and manage their menus, while customers can browse nearby restaurants, add food items to a cart, and place orders.

Once an order is placed, the restaurant confirms and assigns a delivery person. Customers can track the delivery partner’s live location and estimated delivery time. The system includes a unified cart and payment flow across the application.

Additional features include menu browsing, ratings and reviews, saved addresses, order history, and restaurant dashboards for menu and order management. The goal is to provide a simple and efficient food ordering and delivery experience.
